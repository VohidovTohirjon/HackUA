# CareBridge AI

CareBridge AI is a low-bandwidth AI care coordination prototype for rural chronic-care teams.

The demo is built for the HackUA Social Innovation - Rural Healthcare Access track and the Best Use of AI for Public Good prize. It uses synthetic patient data and simulated AI logic to show how SMS and voice-style check-ins can help care teams identify transportation, broadband, medication, appointment, and symptom-related barriers earlier.

## Run locally

```bash
npm install
npm run dev
```

Then open the localhost URL printed by Vite.

## Build

```bash
npm run build
```

## Safety

CareBridge AI does not diagnose, prescribe, change treatment, or replace healthcare professionals. It supports care coordination, barrier detection, summarization, prioritization, and human follow-up.
